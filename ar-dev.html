AR Platform Github Repo

Product Description:

A web platform to allow everyone to create their own Web AR experience without writing any code

AR Platform Demo

Key Technologies Used:

EJS for front-end view engine
Node+Express for backend
MongoDB for database
AR.js: For Augmented Reality
three.js: 3D Model Preview

Code Structure: Model View Controller (MVC)

MVC structure limits code duplication as it separates data and business logic

Specific Technology

User Auth:

Json Web Token(JWT) stored to client with cookie-parser
bcrypt package for user password hash
nodemailer package for user email confirm and password reset

AR/3D

AR.js is used for activating AR content on marker. The library is chosen for its cross-browser compatibility.
three.js is used to preview 3d model and to simplify AR assets transform editing by using its built-in transform control
